Today, I am teaching you how to use the WalServeUs HTTP server, developed by hppavilion1. Your local installationThe folder you downloaded containing all the files of WalServeUs should contain the following files:

  • A folder named config
  • A folder named errors
  • A folder named Help (for beginners) that you will have clicked on to arrive here
  • A folder named logs
  • A folder named pages
  • A file named .gitattributes (a file that has to do with the way this was written)
  • A file named .gitignore (ditto)
  • A file named README.md (this is the more technical version of what you are currently reading)
  • A file named WalServeUs.py (the file that everything runs on)
If you see any others, don't worry. This is probably just due to me forgetting. They probably don't matter to you anyways, as if they did I would have remembered them :P.
